ğŸ”‘ Secrets (Sensitive Data)

    echo -n 'text' | base64: Manually encode a string for a YAML Secret.

    kubectl apply -f secret.yml: Create a secret from a manifest.

    kubectl create secret generic [name] --from-literal=KEY=VALUE: Create a secret directly (auto-encodes to base64).

    kubectl describe secret [name]: See metadata and size of data (hides the actual value).

    kubectl get secret [name] -o jsonpath='{.data.key}' | base64 --decode: Decode and view the actual password.
